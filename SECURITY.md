# Security Policy

## Supported Versions

Currently, only the latest version of this project is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please report it to us by following these steps:

1.  **Do not** open a public issue.
2.  Send an email to security@healthtracker.example.com with a detailed description of the vulnerability.
3.  Include any relevant information, such as:
    - Steps to reproduce the vulnerability.
    - Potential impact.
    - Any suggested fixes.

We will acknowledge your report as soon as possible and work to address the issue promptly. We ask for your patience and cooperation as we investigate and resolve the vulnerability.
